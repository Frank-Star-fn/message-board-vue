# Message Board by Vue
## update
增加了绿色主题可供选择，新增删除留言功能。

## 简介
一个简易留言板。\
技术栈: \
前端: Vue3 + Tailwind CSS + Less \
后端: Node.js + Express\
数据库: MySQL \
UI界面参考
[Xoyondo留言板](https://xoyondo.com/zh/create-personal-message-board), 推荐页面大小为800x1000.

## Project Setup
```sh

```

### Run
前端：
```sh
npm run dev
```

后端：
```sh
node server.js
```
